Instruções para o Jogo de Forca !

Objetivo do Jogo: No jogo da forca, o objetivo é adivinhar a palavra escolhida antes que o desenho do enforcado esteja completo. Cada letra correta revelada ajuda a formar a palavra, enquanto cada letra errada resulta na construção do enforcado. O desafio é adivinhar a palavra corretamente antes que o enforcado seja totalmente desenhado.

Processo de Adivinhação: O usuário faz um palpite inicial digitando uma letra. O programa então verifica se o palpite está correto.

Feedback do Palpite: Se o palpite do jogador estiver incorreto, o programa colocara uma parte do corpo do enforcado

Interação: O usuário pode selecionar letras até adivinhar corretamente a palavra correta ou o enforcado estar completo.

Fim do Jogo: o jogo acaba quando o jogador acerta a palavra ou o enforcado é completo.

Registro de Tentativas: Durante o jogo, o programa mantém o controle das letras selecionadas. Letra verde mostra q acertou e roxa que está incorreta. Divirta-se!
